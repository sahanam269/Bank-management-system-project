# Bank-management-system-project
📌 Bank Management System (Python OOP)  This project is a simple Bank Management System developed using Python and Object-Oriented Programming (OOP) concepts. It simulates basic banking operations and demonstrates how real-world banking systems manage user accounts.  
.

🚀 Features

Create a new bank account

Login using account number and PIN

Deposit and withdraw money

Check account balance

Secure account handling using OOP principles

🛠️ Technologies Used

Python

OOP Concepts (Classes, Objects, Encapsulation, Methods)

📚 Concepts Applied

Class and Object creation

Constructor (__init__)

Data encapsulation

Conditional statements and loops

🎯 Purpose

This project was created to practice and understand OOP
